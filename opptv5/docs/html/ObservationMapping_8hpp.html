class  abt::ObservationMapping
 

Detailed Description

Contains the abstract base class ObservationMapping.

This class defines a mapping of observations following a previous belief and action to subsequent beliefs in the belief tree - each entry in such a mapping is an edge in the tree, which is represented by an ObservationMappingEntry.

The purpose of this abstract class is to allow different observations to be grouped together in customizable ways.
