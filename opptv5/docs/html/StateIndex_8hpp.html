class  abt::StateIndex
 

Detailed Description

Defines an interface allowing states to be indexed in a custom manner.

Currently the only provided implementation of this interface is the RTree, which is a wrapper for the RTree class within libspatialindex.

The RTree-based approach should be sufficient for most purposes, but you can easily make your own implementation of this interface if need be.
