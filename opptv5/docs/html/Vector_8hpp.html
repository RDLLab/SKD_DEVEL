class  abt::Vector
 

Detailed Description

An abstract extension of the Point class for points that can be easily be represented as real vectors.

This is primarily used to provide an interface for storing states (via VectorState, which is currently just typedef of Vector) within an RTree. This allows for efficient lookup via range-based queries.
