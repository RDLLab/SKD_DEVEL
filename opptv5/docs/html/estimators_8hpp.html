Detailed Description

Defines an interface for how backpropagation will be handled; this is done by providing a custom way to estimate the value of a belief based on the q-values of its actions.

This file also provides a few basic implementations of this interface via three core functions; these allow for an estimate of the value of the node via one of: average - the visit-weighted average of its action children max - the maximum value of its action children robust - the value of the action child with the greatest number of visits
